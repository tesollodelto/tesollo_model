# tesollo_model
This repository provides the official simulation models for Tesollo's Delto Gripper/Hand series.

📂 Repository Structure

The repository is organized by model series, containing the necessary assets and physics configurations for each gripper/hand.

- dg3fm/: Models for the DG-3F-M series (12 DOF).

- dg4f/: Models for the DG-4F series (18 DOF).

- dg5f/: Models for the DG-5F series (20 DOF).

- dg5fs/: Models for the DG-5F-S series (Available in both 20 DOF and 15 DOF versions).
